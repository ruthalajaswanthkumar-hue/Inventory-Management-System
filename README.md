# Inventory Management System (Python – Tkinter)
 Project Overview
The Inventory Management System is a GUI-based desktop application developed using **Python and Tkinter**.
It helps manage products, categories, suppliers, employees, billing, and stock efficiently.
This project was built as part of my academic learning to strengthen concepts in **Python, OOPs, GUI development, and database handling**.

Technologies Used

*  Python 
*  Tkinter  (GUI)
*  SQLite  (Database)
*  OOP Concepts 
*  CRUD Operations 

-Features

* Secure Login System 
* Product Management (Add / Update / Delete)
* Category & Supplier Management
* Employee Management
* Billing System
* Dashboard with stock overview
* Image support for GUI
* SQLite database integration
